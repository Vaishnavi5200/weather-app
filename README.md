# 🌦️ Weather App

A modern weather application built using HTML, CSS, JavaScript, and OpenWeatherMap API.

## 🚀 Features

- Search weather by city name
- Real-time weather updates
- Temperature, humidity, and wind speed
- Responsive design for mobile and desktop
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
